### Requerimientos
- Python 3.10 o superior
- `pip install bcrypt`

### Instalación
1. Clonar el repositorio (SSH o HTTPS)
    ```bash
    git clone git@github.com:nicoarbio/hrconnect.git
    ```
    ```bash
    git clone https://github.com/nicoarbio/hrconnect.git
    ```

1. Entrar al repositorio 
   ```bash
   cd hrconnect
   ```
